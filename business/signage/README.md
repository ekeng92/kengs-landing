# Property Signage

Printable signs for Keng's Landing. Design in Canva/Figma, export as PDF, print and laminate.

## Signs Needed

### Indoor
- [ ] **Propane Heater** — Operating instructions for living room heater, safety warnings
- [ ] **Air Conditioner** — Thermostat instructions, preferred temp ranges
- [ ] **Septic Safe Awareness** — What NOT to flush/pour down drains (no grease, wipes, chemicals)
- [ ] **Wi-Fi Info** — Network name and password (update each season if rotated)

### Outdoor
- [ ] **Hot Tub Rules** — Max occupancy, no glass, shower first, cover when done, temperature limits
- [ ] **Pond** — No swimming, supervise children, catch-and-release fishing rules (if applicable)
- [ ] **Outdoor Fires** — Fire pit rules, county burn restrictions, extinguish before bed
- [ ] **Wooded Acres** — Trail info, wildlife awareness, tick/snake precautions, stay on paths
- [ ] **Parking** — Where to park, how many vehicles

## Style Guide
- Keep consistent branding (Keng's Landing logo/name)
- Warm, welcoming tone — not corporate
- Laminate for durability
- Standard letter size (8.5x11) or half-sheet where appropriate
