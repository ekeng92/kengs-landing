# Monthly Booking Log

> Copy this template to `finances/2026/revenue/YYYY-MM-bookings.ignore.csv` and fill in real data.

## CSV Format

```csv
date_booked,platform,guest_name,series,check_in,check_out,nights,nightly_rate,cleaning_fee,platform_payout,platform_fees,notes
```

## Column Definitions

| Column | Description |
|--------|-------------|
| `date_booked` | Date the reservation was confirmed (YYYY-MM-DD) |
| `platform` | Airbnb, VRBO, Zillow, TurboTenant, Direct |
| `guest_name` | Guest first name or booking ID |
| `series` | 360, Ironwood, or Marlow |
| `check_in` | Check-in date (YYYY-MM-DD) |
| `check_out` | Check-out date (YYYY-MM-DD) |
| `nights` | Number of nights |
| `nightly_rate` | Rate per night charged |
| `cleaning_fee` | Cleaning fee charged to guest |
| `platform_payout` | Actual amount deposited to your bank |
| `platform_fees` | Fees withheld by the platform |
| `notes` | Cancellation, refund, special circumstances |
