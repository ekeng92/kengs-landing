# Monthly Expense Log

> Copy this template to `finances/2026/expenses/YYYY-MM-expenses.ignore.csv` and fill in real data.

## CSV Format

```csv
date,series,category,vendor,description,amount,payment_method,receipt,notes
```

## Column Definitions

| Column | Description |
|--------|-------------|
| `date` | Transaction date (YYYY-MM-DD) |
| `series` | 360, Ironwood, Marlow, or LLC-General |
| `category` | Schedule E category (see list below) |
| `vendor` | Who you paid (Home Depot, Cleaner name, etc.) |
| `description` | Brief description of what was purchased/done |
| `amount` | Dollar amount |
| `payment_method` | Credit card, debit, cash, check, Venmo |
| `receipt` | Y/N — whether receipt is saved |
| `notes` | Additional context |

## Schedule E Categories

Use these exact values in the `category` column:

- `mortgage_interest`
- `property_taxes`
- `insurance`
- `repairs_maintenance`
- `supplies`
- `utilities`
- `cleaning_turnover`
- `platform_fees`
- `professional_services`
- `advertising`
- `travel_mileage`
- `depreciation`
- `other`
