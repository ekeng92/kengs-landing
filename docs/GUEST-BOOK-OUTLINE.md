# Guest Book — Content Outline

Source file: `docs/Kengs_Landing_Guest_Book.docx`

## Suggested Sections

- [ ] **Welcome** — Brief intro, property story, what makes it special
- [ ] **Check-in / Check-out** — Arrival time, key/lockbox instructions, departure checklist
- [ ] **Wi-Fi & Entertainment** — Network, password, streaming apps, TV instructions
- [ ] **Kitchen** — Appliances guide, coffee maker, what's stocked, dishwasher instructions
- [ ] **Climate Control** — A/C thermostat, propane heater in living room, ceiling fans
- [ ] **Hot Tub** — How to use, cover on/off, temperature, chemicals (if guest-managed)
- [ ] **Outdoor Spaces** — Fire pit, pond, wooded acres, deck/patio
- [ ] **Laundry** — Washer/dryer location and instructions
- [ ] **Trash & Recycling** — Pickup schedule, where bins go, what's recyclable
- [ ] **Septic System** — What NOT to flush (critical for rural properties)
- [ ] **Emergency Info** — Local hospital, fire dept, police non-emergency, property manager contact
- [ ] **Local Recommendations** — Restaurants, groceries, attractions, hikes, breweries
- [ ] **House Rules** — Quiet hours, smoking policy, pet policy, max occupancy, parking
- [ ] **Departure Checklist** — Strip beds, start dishwasher, take out trash, lock up, thermostat setting
